# v0.10.14 — Update Prompt Wording

**Date:** 2026-05-25

## Highlights

- Changed the welcome update prompt to `what's new in this update?`.
- Kept the prompt routed to cumulative pending release notes.
- Added the new phrase to the release-notes skill routing examples.

## Why This Patch Exists

Production validation confirmed the full cycle works: welcome detects a new
stable release, the suggested release-notes question renders cumulative pending
notes, and `update my Mirror` updates the runtime. The remaining issue was tone.
`show pending release notes` is accurate but too technical for a final-user
welcome.

## What Changed

The welcome now renders:

```text
Ask: "what's new in this update?" or "update my Mirror"
```

The release-notes skill recognizes that phrase and continues to use the pending
release-notes flow.

## Next Horizon

This closes the release-awareness wording loop. Future update UX can build on
this by making the welcome actions more command-like or clickable in runtimes
that support richer UI affordances.
