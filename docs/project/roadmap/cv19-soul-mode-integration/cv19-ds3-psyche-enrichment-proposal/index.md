# CV19.DS3 — Psyche Enrichment Proposal

**Status:** ✅ Done

**User-visible outcome:** The user can see how review material might become Self, Shadow, Ego, or persona identity without anything being written.

## Scope

- Add proposal-only Soul Mode enrichment surfaces.
- Support `self`, `shadow`, `ego`, and `persona` targets.
- Show target, origin, current material, proposed material, and rationale.
- Always state `proposal only — no identity changed`.
- Exclude journey identity.

## Acceptance Behavior

Given the user asks how review material could be integrated, Mirror renders an enrichment proposal.

Given a proposal renders, no identity entry is changed.

Given persona is targeted, a persona key is required.

## Non-goals

- No identity mutation.
- No journal save.
- No journey identity proposal.
