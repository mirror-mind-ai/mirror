# CV17.DS2 — Living Field And Possible Listenings

**Status:** ✅ Done

**Placement:** CV17 first transition story after Soul Mode entry

**User-visible outcome:** After the user answers the entry question, Mirror can conduct a free opening conversation, recognize living matter, and render situated Possible Listenings at the right moment.

---

## Why This Exists

The user will not operate Soul Mode as a workflow. They will answer “how is your day going today?” naturally. Soul Mode intelligence begins in the capacity to listen for when ordinary description becomes ritual matter.

The transition surface is not a menu. It is a situated invitation:

```text
Soul Mode
╭────────────────────────────────────────╮
│   ✧  POSSIBLE LISTENINGS               │
│                                        │
│   ✦ Self Voice                         │
│     recognize the principle that wants │
│     to be preserved                    │
│                                        │
│   ◐ Shadow Voice                       │
│     listen to the part that wants to   │
│     be necessary                       │
│                                        │
│   ☉ Wisdom Voice                       │
│     be crossed by an idea about value  │
│     and recognition                    │
╰────────────────────────────────────────╯
```

---

## Scope

- Define the Listening To The Living Field phase in the Soul Mode skill contract.
- Define maturity markers for when Possible Listenings should appear.
- Render a Possible Listenings surface at the end of the response when enough living matter is present.
- Keep descriptions situated in the user’s last turn.
- Allow Mirror to suggest a dominant listening without removing user choice.
- Preserve natural-language interaction; no rigid command menu.

---

## Non-goals

- No active rite implementation beyond preparing the selection boundary.
- No persistent fruit state.
- No journal persistence.
- No full LLM classifier unless the story plan later chooses that route.
- No expectation that every user answer produces Possible Listenings.

---

## Acceptance Behavior

Given Soul Mode is active and the user describes their day, Mirror can continue a normal reflective conversation without forcing a rite too early.

Given a user turn reveals living matter such as a touched center, emerging shadow, threatened value, search for clarity, request for beauty, or strong dispersion, Mirror can offer Possible Listenings.

Given Possible Listenings are rendered, each option is phrased in relation to what the user brought rather than as generic voice copy.

Given no living matter has emerged, Mirror should ask or reflect rather than show the card prematurely.
