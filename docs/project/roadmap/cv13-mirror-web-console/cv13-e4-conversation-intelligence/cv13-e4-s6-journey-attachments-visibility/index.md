# CV13.E4.S6 — Journey attachments visibility

**Status:** ✅ Done
**Epic:** CV13.E4 — Conversation Intelligence
**Release target:** v0.14.0

---

## User-visible outcome

The Workspace for a selected journey shows the journey's attached reference material in a read-only Attachments tab.

---

## Scope

- Surface attachments for the selected journey in Workspace.
- Show attachment name, description/content preview, content type, journey, tags, and created date where available.
- Add an attachments metric for the selected journey.
- Keep the story read-only.

---

## Non-goals

- No attachment upload.
- No attachment editing.
- No attachment deletion.
- No full attachment detail page unless already supported by generic object routing.
- No LLM calls.
